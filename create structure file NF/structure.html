Описание работы

  • Сначала нужно создать ветки, только потом загружать слайды
  • Слайды можно загружать множественно, т.е. выделить сразу все превью, они все подгрузятся
  • Перетаскиванием слайдов в ячейки формируем содержимое веток. Удалить слайд из ветки пока нельзя (только всю ветку)
  • По клику add event начинается формирование события. Порядок такой - сначала клик по add event, потом - по радиокнопке, напротив каждой ветки / слайда. Затем обязательно нажать кнопку ок.
  • По завершении создания всех событий нажать кнопку structure, появится строка json. Ее нужно скорировать и отформатировать табуляцию.
